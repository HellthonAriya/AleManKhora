# اِل من خورا (AleManKhora)

یک بازی استراتژیک آنلاین کامل و چندنفره، الهام‌گرفته از **Quoridor** (مثل wrongway.app).
در هر نوبت یا مهره‌ات را حرکت می‌دهی یا یک **دیوار** می‌سازی تا راه حریف را ببندی.
هدف: پیش از حریف به آن‌سوی صفحه برسی.

A complete, real-time, multiplayer Quoridor-style strategy game with guest play,
accounts, matchmaking, private invites, AI practice, an ELO leaderboard, rich
Canvas graphics, per-game customization, and a full admin panel.

---

## ✨ امکانات

- 🎲 **بازی تصادفی (Matchmaking)** — همگام‌سازی آنی با حریفی هم‌سطح.
- 🤝 **دعوت دوست** — اتاق خصوصی با کد ۶ حرفی و لینک دعوت.
- 🤖 **بازی با هوش مصنوعی** — سه سطح سختی (آسان/متوسط/سخت).
- 👤 **مهمان یا حساب کاربری** — هم بدون ثبت‌نام بازی کن، هم با حساب امتیاز جمع کن.
- 🏆 **جدول رتبه‌بندی ELO** و صفحهٔ پروفایل با آمار و تاریخچهٔ بازی‌ها.
- 🎨 **سفارشی‌سازی هر بازی** — اندازهٔ صفحه (۵×۵ تا ۱۱×۱۱)، تعداد دیوارها،
  تم رنگی صفحه، و رنگ هر بازیکن، با پیش‌نمایش زنده.
- 🖼 **گرافیک Canvas** — انیمیشن حرکت مهره، افکت ساخت دیوار، هایلایت حرکت‌های مجاز،
  پیش‌نمایش زندهٔ جای دیوار، و شش تم.
- 💬 گفتگوی درون‌بازی، تماشاگر، اتصال مجدد پس از قطعی، و بازی مجدد (Rematch).
- 🛡 **پنل مدیریت کامل** — داشبورد آمار، مدیریت کاربران (مسدودسازی/مدیر/حذف/تغییر رمز)،
  فهرست بازی‌ها و پیکربندی سراسری سایت.

## 🛠 پشتهٔ فنی

| بخش | فناوری |
|-----|--------|
| سرور | Node.js + Express |
| بلادرنگ | Socket.IO |
| پایگاه‌داده | SQLite (`better-sqlite3`) |
| احراز هویت | JWT (کوکی httpOnly) + bcrypt |
| کلاینت | Vanilla JS (ES Modules) + HTML5 Canvas — بدون مرحلهٔ build |
| موتور بازی | یک نسخهٔ واحد در `public/js/engine.js`، مشترک بین سرور و مرورگر |

## 🚀 اجرا

```bash
npm install          # نصب وابستگی‌ها
npm run seed         # (اختیاری) ساخت کاربر مدیر و چند بازیکن نمونه
npm start            # اجرای سرور روی http://localhost:3000
```

- اولین کاربری که ثبت‌نام کند به‌صورت خودکار **مدیر** می‌شود.
- اگر `npm run seed` را اجرا کنی، حساب مدیر آماده است:
  - نام کاربری: `admin` — رمز: `admin123`

متغیرهای محیطی:

- `PORT` — پورت سرور (پیش‌فرض `3000`).
- `JWT_SECRET` — کلید امضای توکن (در محیط تولید حتماً تنظیم کن).

## 📂 ساختار پروژه

```
server/
  index.js          نقطهٔ ورود سرور (Express + Socket.IO)
  db.js             راه‌اندازی SQLite و تنظیمات سراسری
  models.js         مدل کاربر/بازی و محاسبهٔ ELO
  auth.js           احراز هویت JWT و میدلورها
  socket.js         لایهٔ بلادرنگ Socket.IO
  game/
    engine.js       (re-export از public/js/engine.js)
    ai.js           حریف هوش مصنوعی
    manager.js      مدیریت اتاق‌ها، matchmaking و دعوت
  routes/
    api.js          REST API (احراز هویت، پروفایل، رتبه‌بندی)
    admin.js        REST API پنل مدیریت
public/
  index.html        پوستهٔ تک‌صفحه‌ای (SPA)
  css/style.css     سیستم طراحی و تم‌ها
  js/
    app.js          روتر و راه‌اندازی
    engine.js       موتور Quoridor (منبع واحد، مشترک با سرور)
    board.js        رندر Canvas صفحه
    components.js    اجزای قابل‌استفادهٔ مجدد (سفارشی‌ساز بازی)
    core.js          ابزارها: DOM، state، API، toast، modal
    views/           صفحات: home, auth, lobby, game, leaderboard, profile, admin
```

## 📜 قوانین بازی (خلاصه)

- هر بازیکن یک مهره دارد که از یک لبهٔ صفحه شروع می‌شود و باید به لبهٔ مقابل برسد.
- در هر نوبت یکی از این دو کار را انجام بده:
  1. **حرکت** مهره به یک خانهٔ مجاور (با امکان پرش از روی حریف).
  2. **ساخت دیوار** بین خانه‌ها برای بستن مسیر حریف.
- دیوار را نمی‌توان طوری گذاشت که مسیر هیچ بازیکنی به‌طور کامل بسته شود
  (با جست‌وجوی مسیر بررسی می‌شود).
- هر بازیکن تعداد محدودی دیوار دارد.
